WorldProno2014
============

Football WorldCup Pronostics 2014 App

## Production
To See App in production:
[Live version](http://worldprono2014.herokuapp.com/)


Version
----

0.1

Tech
-----------

WorldProno use following tech to work properly :

* [node.js] - evented I/O for the backend
* [Express] - fast node.js network app framework
* [Twitter Bootstrap] - great UI boilerplate for modern web apps
* [AngularJS] - duh 
* [jQuery] - duh 

Installation
--------------

* installer nodejs 
* installer [Grunt] via une fenetre DOS 
```sh
npm install -g grunt-cli
```
_le -g install sur le système (et non en local) et est disponible dans tous tes répertoires (donc tous les developpements)_

* installer [Bower] 
```sh
npm install -g bower
```
